# Appscrip Task Candidate

Responsive product listing page demo built with Next.js and server-side rendering.

## Stack

- Next.js
- React
- Plain CSS

## Features

- Server-side rendering via `getServerSideProps`
- Responsive layout for mobile, tablet, and desktop
- Category filter interaction
- SEO metadata, canonical URL, Open Graph tags, and JSON-LD schema
- SEO-friendly local image asset names and descriptive alt text
- Standalone HTML/CSS version in [index.html](/C:/Users/saura/OneDrive/Desktop/New%20folder%20(4)/index.html)

## Local Development

```bash
npm install
npm run dev
```

## Production Build

